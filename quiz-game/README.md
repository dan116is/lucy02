# אלוף הידע 🎖️

משחק טריוויה ומבחני סימולציה בעברית (RTL), בהשראת אפליקציות לתרגול מבחני
מיון. ללא תלות חיצונית, רץ ישירות בדפדפן ובמובייל.

## אפליקציה לאייפון (PWA) 📱

המשחק הוא **Progressive Web App** מלא וניתן להתקנה כמו אפליקציה רגילה:

- **התקנה למסך הבית** — באייפון: פתח ב־Safari ‹ כפתור **שיתוף** ‹ **הוסף למסך הבית**.
  נפתח במסך מלא, בלי שורת דפדפן, עם אייקון ומסך פתיחה משלו.
- **עבודה לא־מקוונת (offline)** — `service worker` שומר את כל קבצי האפליקציה
  במטמון, כך שהמשחק נטען ופועל גם בלי חיבור לאינטרנט.
- **אייקונים ומסכי פתיחה** — אייקון מדליית זהב בכל הגדלים (כולל maskable)
  ומסכי פתיחה ייעודיים לכל דגמי האייפון הנפוצים, בתיקיית `icons/`.
- **תמיכה ב־safe-area** לנוטש/Dynamic Island ובשפת העיצוב של iOS.

קבצי ה־PWA: `manifest.webmanifest`, `sw.js`, `offline.html`, `icons/`.

## מצבי משחק

- **סימולציית מבחן** — 20 שאלות בזמן קצוב, בלי חשיפת תשובות באמצע, ציון וסקירת
  טעויות בסוף — בדיוק כמו במבחן אמיתי.
- **מצב למידה** — משוב והסבר מיד אחרי כל שאלה; שאלות שטעית בהן חוזרות שוב עד
  שאתה שולט בכל החומר.
- **משחק קלאסי** — בליץ של 90 שניות, 3 חיים, בונוס זמן ורצף, מערכת דרגות
  (טוראי → … → אלוף) ושיא אישי.

## מאגר השאלות

**4,050 שאלות** בעשרה נושאים (ישראל, תנ״ך וחגים, גאוגרפיה, מדע וחלל, ספורט,
ידע כללי, עברית, חשבון, חשיבה כמותית). המאגר נבנה ב־`data.js` משילוב של
**250 שאלות טריוויה כתובות ומאומתות** ומחוללים עובדתיים (בירות, יסודות
כימיים, ספרות רומיות) + תרגול כמותי. כדי להוסיף שאלות ידניות, ערוך את המערך
`CURATED` או הוסף מחולל חדש.

> השאלות נכתבו/נוצרו במיוחד למשחק זה ואינן לקוחות מבנק שאלות קנייני כלשהו.

## הרצה מקומית

```bash
python3 -m http.server 8000
# פתח http://localhost:8000/quiz-game/
```

הניקוד והדרגה נשמרים מקומית (`localStorage`) במכשיר בלבד.
